BootstrapModalMixin for React components

A mixin for components which act as Bootstrap modals, providing:

  • Registration of the component as a modal at the appropriate point in the component lifecycle
    • Components are responsible for providing full modal contents in their own render method
  • Properties for modal configuration, with default values as per Bootstrap's defaults
    • backdrop
    • keyboard
    • show
    • remote
  • Registration and removal of event handlers with React-style naming conventions when any of the following properties are part of the modal component's prototype or are passed in as component props (in that order)
    • handleShow
    • handleShown
    • handleHide
    • handleHidden
  • A convenience method for rendering a × close button: renderCloseButton()
